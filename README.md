# Welcome to NetFromScratch

Project with the goal of creating a .NET Core web application from scratch to deployment in VS Code.

This is an ASP.NET Core project. [Learn what's new](https://go.microsoft.com/fwlink/?LinkId=518016)

## This application consists of:

*   Sample pages using ASP.NET Core MVC
*   [Bower](https://go.microsoft.com/fwlink/?LinkId=518004) for managing client-side libraries
*   Theming using [Bootstrap](https://go.microsoft.com/fwlink/?LinkID=398939)

## Building and running NetFromScratch

<dl>
  <dt>Clone the NetFromScratch project to a local git repo</dt>

  <dt>Open a cmd window at the root project location</dt>

  <dt>Restore Nuget Packages:</dt>
  <dd>dotnet restore</dd>

  <dt>Build NetFromScratch:</dt>
  <dd>dotnet build</dd>
  
  <dt>Run the application:</dt>
  <dd>dotnet run</dd>

  <dt>Project is now running at location Localhost:5000</dt>

</dl>
